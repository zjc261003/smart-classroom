# smart-classroom
## 主要功能
### 智能门禁
##### 一个非常常见的东西啦，目前某些学校里都有。
### 呼唤学生
##### 老师能够通过某平台向教室内发送消息
### 远程控制
##### 老师能通过某平台向教室内的电器发送状态消息，从而改变电器状态（说得太牛了些）
### 课堂互动
##### 能够快速采集课堂互动的数据
## 新情报
### 手势识别 & 姿态识别
##### 由于手势识别和姿态识别不是很稳定，决定不将这项功能加入智学天地
